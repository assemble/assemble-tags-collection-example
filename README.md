# assemble-tags-collection-example 

> Example of how to create a collection with assemble.

## Get started

Clone the project

```bash
$ git clone https://github.com/assemble/assemble-tags-collection-example.git
```


## Usage

Install dev dependencies and run `assemble`

```bash
$ npm i -d && assemble
```
